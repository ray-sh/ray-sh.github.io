# Python 变量与函数通俗详解

在 Python 中，变量和函数是构建代码的两大基础核心——变量负责“存储数据”，就像我们生活中的“容器”；函数负责“封装功能”，就像工厂里的“生产线”。理解这两个概念，就能看懂大部分简单的 Python 代码。下面从“是什么→为什么用→怎么用”的逻辑，由浅入深拆解说明。

## 一、变量：存储数据的“容器”

### 1. 变量是什么？（通俗理解）

变量就是给数据起的“名字”，把数据存到变量里，后续用这个名字就能直接找到对应的数据。就像你用“冰箱”存放食物——“冰箱”是名字（变量名），里面的“苹果、牛奶”是数据；后续想拿食物，直接找“冰箱”就行，不用记住每样食物的具体位置。

比如在 Python 里写：`age = 18`，这里的 `age` 就是变量，`18` 是存进去的数据。之后想用到“18”这个数据，直接写 `age` 就可以。

### 2. 为什么需要变量？

没有变量的话，重复使用数据要反复写原始值，既麻烦又容易错。比如计算“小明 3 年后的年龄”和“5 年前的年龄”：

不用变量：`print(18 + 3)`、`print(18 - 5)`（如果要改年龄，两处的 18 都要改）；

用变量：`age = 18`、`print(age + 3)`、`print(age - 5)`（改年龄时，只改 `age = 18` 这一处就行）。

### 3. 变量的使用规则（必记）

- **命名规则**：只能由字母、数字、下划线组成，不能以数字开头（比如 `age18` 合法，`18age` 非法）；区分大小写（`Age` 和 `age` 是两个不同变量）；不能用 Python 自带的关键字（比如 `print`、`if` 不能当变量名）。

- **赋值符号**：用 `=` 给变量存数据（注意：`=` 是“赋值”，不是“等于”），比如 `name = "小明"`（存字符串）、`score = 95.5`（存小数）、`is_pass = True`（存布尔值：是/否）。

- **可修改性**：变量里的数据可以随时替换，比如 `age = 18` 之后，再写 `age = 19`，变量 `age` 就变成 19 了。

### 4. 变量的简单示例

```python

# 定义 3 个不同类型的变量
name = "小红"  # 字符串变量（存文本，用引号包裹）
height = 165.5  # 浮点数变量（存小数）
has_book = True  # 布尔变量（存 True/False，代表是/否）

# 使用变量：打印变量的值
print("名字：", name)
print("身高：", height)
print("是否有书：", has_book)

# 修改变量的值
height = 166.0
print("更新后的身高：", height)
```

运行结果：

```Plain Text

名字： 小红
身高： 165.5
是否有书： True
更新后的身高： 166.0

```


## 二、函数：封装功能的“生产线”

### 1. 函数是什么？（通俗理解）

函数是把一段“重复使用的代码块”打包起来，起一个名字——后续想执行这段代码，不用重新写，直接喊这个名字就行。就像工厂里的“果汁生产线”：把“苹果”（输入数据）放进去，生产线（函数）自动完成“清洗→榨汁→装瓶”（代码逻辑），最后出来“苹果汁”（输出结果）；下次想做橙汁，只要换“橙子”（输入），再启动生产线（调用函数）就行。

Python 里有两种函数：

- **内置函数**：Python 自带的“现成生产线”，比如 `print()`（打印内容）、`len()`（计算长度），直接用就行，不用自己写。

- **自定义函数**：自己打包的“专属生产线”，比如写一个“计算两数之和”的函数，后续计算任何两个数的和，都能直接用。

### 2. 为什么需要函数？

如果没有函数，重复的代码要写很多遍，代码又长又难维护。比如想计算“10+20”“30+40”“50+60”：

不用函数：要写 3 遍类似的计算代码；

用函数：写 1 遍“求和代码”打包成函数，之后调用 3 次函数就行，改逻辑时也只改 1 处。

### 3. 函数的核心要素（必记）

自定义函数的格式固定，就像搭生产线要先画“设计图”，核心要素包括：

- **def 关键字**：告诉 Python“我要定义函数了”，固定写在最前面。

- **函数名**：给函数起的名字，规则和变量名一样（字母/数字/下划线，不以数字开头），比如 `add_num`（求和）、`calc_area`（计算面积）。

- **参数（括号里的内容）**：函数的“输入数据”，就像生产线的“原料入口”。比如求和函数需要两个数，括号里就写两个参数：`def add_num(a, b):`（a 和 b 就是要输入的两个数）。

- **冒号和缩进**：`def` 最后要加冒号，后面的代码块要缩进（一般 4 个空格），代表“这是函数里的代码”。

- **return 语句**：函数的“输出结果”，就像生产线的“成品出口”。比如求和函数计算完 a+b 后，用 `return a + b` 把结果返回出去；如果没有 `return`，函数默认返回 `None`（空值）。

### 4. 函数的使用步骤（自定义+调用）

使用自定义函数分两步：先“定义函数”（搭生产线），再“调用函数”（启动生产线）。

#### 示例 1：定义一个“求和函数”

```python

# 第一步：定义函数（搭生产线）
def add_num(a, b):  # a、b 是参数（原料入口）
    sum_result = a + b  # 函数内部的计算逻辑（生产线流程）
    return sum_result  # 返回结果（成品出口）

# 第二步：调用函数（启动生产线）
result1 = add_num(10, 20)  # 输入 10 和 20，接收返回的结果
result2 = add_num(30, 40)  # 再次调用，输入不同参数
print("10+20=", result1)
print("30+40=", result2)
```

运行结果：

```Plain Text

10+20= 30
30+40= 70
```

#### 示例 2：无参数、无返回值的函数

如果函数不需要输入（无参数）、不需要输出（无返回值），比如“打印欢迎语”，格式可以简化：

```python

# 定义无参数、无返回值的函数
def print_welcome():
    print("----------------")
    print("  欢迎使用本程序  ")
    print("----------------")

# 调用函数（不用传参数，也不用接收结果）
print_welcome()
print_welcome()
```

运行结果：两次打印欢迎语，代码简洁很多。

### 5. 内置函数的简单使用

内置函数不用定义，直接调用，比如：

```python

# print()：打印内容（无返回值，只做操作）
print("Hello Python")

# len()：计算字符串/列表的长度（有返回值）
text = "abcde"
length = len(text)  # 输入 text，返回长度 5
print("字符串长度：", length)

# max()：找最大值（有返回值）
num_list = [12, 34, 5]
max_num = max(num_list)
print("最大值：", max_num)
```

## 三、变量与函数的关联：数据的“流转”

变量和函数不是孤立的，而是配合使用的——变量可以作为函数的“输入参数”，函数的“返回结果”也可以存到变量里，实现数据的流转。

举个综合例子：计算两个学生的总分（语文+数学）

```python

# 1. 定义变量：存储两个学生的语文、数学成绩
student1_chinese = 90
student1_math = 85
student2_chinese = 88
student2_math = 92

# 2. 定义函数：计算单科总分（输入两个科目成绩，返回总分）
def calc_total(subject1, subject2):
    total = subject1 + subject2
    return total

# 3. 调用函数：用变量作为参数，结果存到新变量
student1_total = calc_total(student1_chinese, student1_math)
student2_total = calc_total(student2_chinese, student2_math)

# 4. 打印结果：使用变量
print("学生1总分：", student1_total)
print("学生2总分：", student2_total)
```

运行结果：

```Plain Text

学生1总分： 175
学生2总分： 180
```

这里的流转逻辑：变量（成绩）→ 输入函数（计算）→ 函数返回结果→ 存到新变量（总分）→ 打印变量（输出）。

## 四、新手避坑指南（常见问题）

- **变量未定义就使用？** 比如先写 `print(age)`，再写 `age = 18`，会报错。解决：先给变量赋值，再使用。

- **函数调用时参数数量不匹配？** 比如 `add_num(a, b)` 定义了 2 个参数，调用时只写 `add_num(10)`，会报错。解决：调用时传入的参数数量和定义时一致。

- **函数内部的变量不能外部使用？** 比如函数里定义的 `sum_result`，在函数外面直接写 `print(sum_result)` 会报错。原因：函数内部的变量是“局部变量”，只能在函数里用；外部要用，需要用 `return` 把结果返回出去，再存到外部变量。

- **忘记写函数的冒号或缩进？** 比如 `def add_num(a, b)` 没加冒号，或后面的代码没缩进，会报错。解决：严格按照格式写，`def` 结尾加冒号，函数体缩进 4 个空格。

## 五、核心逻辑一句话总结

变量是“数据容器”，负责存储和传递数据；函数是“功能封装”，负责重复执行特定逻辑；两者配合，就能实现“数据输入→逻辑处理→结果输出”的完整流程，让 Python 代码更简洁、更易维护。
